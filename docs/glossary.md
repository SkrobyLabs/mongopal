# Glossary

MongoDB's own vocabulary (database, collection, document, index, aggregation, BSON) is
assumed. These terms are specific to how MongoPal organizes things.

**Folder** — A hierarchical container for organizing *saved connections* in the sidebar.
Folders nest and support drag-and-drop, letting a user group connections by environment
or project. Purely an organizational construct; it has no effect on the connections
themselves.

**Favorite** — A database or collection the user has starred for quick access from the
sidebar, independent of which folder its connection lives in.

**Extended Connection** — The full, encrypted connection record: MongoDB credentials plus
optional SSH tunnel, TLS certificates, and proxy settings. It is distinct from the older
*saved connection* format (a keyring password alongside a plain connection URI), which
MongoPal still reads for backward compatibility. See
[Connection management](features/connection-management.md).

**Saved query** — A named, reusable query (filter, projection, sort) stored per
collection, distinct from the auto-recorded query history.
